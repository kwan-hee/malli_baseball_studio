# 03. 말리 (MALLI) — 캐릭터·페르소나 정의

말리는 두 채널(야구사전, 말리 동화)의 공용 캐릭터. 이 문서가 캐릭터 진실의 원천.
채널별 콘텐츠 규칙은 04_BASEBALL.md / 09_STORYBOOK.md 참조.

주의: 이 문서의 강아지 말리는 **동화 채널의 주인공**이다. 야구사전 채널의 "말리"는
해설자 페르소나(다른 말투·다른 보이스)로 운영돼 왔음 — 04_BASEBALL.md 참조.
야구사전에서 강아지 말리는 썸네일·삽화 마스코트로만 쓴다.

## 캐릭터 기본 (2026-07-07 확정)

- 이름: 말리 (영문 표기: MALLI — 목걸이 태그 텍스트도 "MALLI". 구 캐릭터 시트의 LILY 표기는 폐기)
- 종: 말티푸 (몰티즈+푸들 믹스) 새끼 강아지, 암컷
- 외형: 크림색 곱슬곱슬한 털, 크고 동그란 검은 눈, 검은 코, 분홍색 목걸이에 빨간 리본(나비 모양), 뼈다귀 모양 "MALLI" 이름표. 머리에 빨간 리본 핀 연출 가능.
- 성격: 아주 밝고 맑고 활발함. 호기심 많음.
- 말투: 여성스럽고 귀여운 말투. 새끼 강아지다운 천진함.

## 그림체 (확정)

- 지브리 스타일 일본 애니메이션풍 + 부드러운 수채화 질감
- 캐릭터 시트 원본: images/style_ref/malli_reference.png (기존 파이프라인에서 복사 완료 — Gemini 이미지 생성 시 multimodal 참조 이미지로 함께 전달)
- 모든 말리 이미지는 prompts/malli_character.txt 기준 프롬프트에서 파생. 매번 새로 묘사 금지 (일관성 붕괴 방지).

## 대본 톤 규칙

- 청자를 "여러분" 또는 "친구들" 로 부름
- 한 문장 20자 내외, 나레이션 호흡 단위로 줄바꿈
- 밝고 통통 튀는 리듬. 감탄사·의성어 자연스럽게 사용 ("우와!", "멍멍!")
- 채널 무관 금지: 비속어, 특정 구단·인물 비하, 검증 안 된 사실 단정
