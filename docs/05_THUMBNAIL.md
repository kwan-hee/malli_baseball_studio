# 05. 썸네일 규칙

## 공통

- 규격: 1280x720, 텍스트는 모바일에서 읽히게 대형 배치
- 텍스트: 최대 2줄, 줄당 7자 이내 권장
- 말리 캐릭터 포함 (docs/03_MALLI.md 캐릭터 시트 기준)

## 채널별

| 항목 | 야구사전 | 말리 동화 |
|------|---------|----------|
| 배경 | 야구장·경기 장면 톤 | 동화 일러스트 톤 |
| 텍스트 성격 | 궁금증형 ("보크가 뭐길래?") | 이야기형 (동화 제목 중심) |
| 색 | (TODO: 채널 고정 색 확정) | (TODO) |

## 제작 순서

1. 썸네일 텍스트는 SEO 패키지에서 확정된 것 사용 (새로 짓지 말 것)
2. 배경 이미지 → docs/06_IMAGE.md 규칙으로 생성
3. 텍스트 합성 → CapCut 또는 ffmpeg drawtext (docs/08_FFMPEG.md)
4. 저장: output/{채널}/{편명}/thumbnail.png
