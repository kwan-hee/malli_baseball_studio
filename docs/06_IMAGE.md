# 06. 이미지 생성 규칙

## 원칙

- 생성·보정·캐릭터 유지·배경 수정 전부 **Nano Banana 기본** (2026-07-07 플랫폼 정책 — docs/01_COST_POLICY.md)
- 비용 규칙 우선 확인 (docs/01_COST_POLICY.md)

- 생성 전 images/ 에 재사용 가능한 이미지 있는지 먼저 확인

## 규격

| 용도 | 크기 |
|------|------|
| 영상 본문 컷 | 1920x1080 (Ken Burns 줌 여유 위해 원본은 더 크게 권장) |
| 쇼츠 | 1080x1920 |
| 썸네일 배경 | 1280x720 이상 |
