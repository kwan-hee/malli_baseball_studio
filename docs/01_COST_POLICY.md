# 01. 비용 정책

## 원칙

1. 무료·로컬 도구 우선. 유료 API는 무료로 불가능할 때만.
2. 초안·실험 단계는 저비용 모델, 최종 렌더만 고품질.
3. 같은 결과물 재생성 금지 — 생성 전 output/ 에 이미 있는지 확인.

## 사용 중인 서비스

Google AI Pro (Google Flow 포함) / Higgsfield Ultra / Nano Banana / Hedra / FFmpeg 자동화

## 영상 생성 플랫폼 정책 (2026-07-07 사용자 확정)

하나의 플랫폼 고정 아님 — 작업 목적에 따라 가장 경제적인 플랫폼 자동 선택.

| 채널 | 기본 플랫폼 | 대안 (불가 시) | 이유 |
|------|------------|---------------|------|
| 말리 동화 | Google Flow — **Veo 3.1** (평상 씬은 Veo 3.1 Fast로 크레딧 절약) | Higgsfield (Seedance) | 대량 제작, 월 Flow Credits 우선 소진, 캐릭터 일관성 > 최고 영상미, AI Pro 구독 혜택 활용 |
| 야구사전 | Higgsfield Ultra | Google Flow | 카메라 무빙, 역동적 연출, 다양한 모델, 고품질 스포츠 표현 |

자동 선택 예시: "말리가 무지개 숲에서 친구를 만난다" → Google Flow / "보크 규칙 설명", "야구 역사" → Higgsfield / "말리 수면동화" → Google Flow.

예외 (플랫폼 변경 허용): Google Flow 사용량 부족 / Higgsfield 장애 / 특정 모델 필요 / 사용자 직접 지정.

## 비용 최적화 순서 (항상 적용)

1. 사용할 플랫폼 결정 → 2. 가장 경제적인 플랫폼 우선 → 3. 같은 결과면 저비용 선택 → 4. 품질 차이 크지 않으면 항상 저비용.

목표: 사용자는 주제만 입력. Claude가 적합성·비용·안정성 기준으로 플랫폼 자동 결정하고 전체 파이프라인 실행.
